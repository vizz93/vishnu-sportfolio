FloorSafe

An All-in-One Incident Management Solution for Gaming Venues

Overview

This tool was developed to assist floor supervisors in gaming venues with efficient incident logging, tracking, and reporting. With a mobile interface for quick, on-the-spot logging and a comprehensive web dashboard for in-depth reporting and incident tracking, the tool significantly enhances operational transparency and response times.

Project Goals

  • Enable Rapid Incident Logging: Create a mobile interface that allows supervisors to log incidents quickly and accurately, minimizing disruption on the gaming floor.

  • Streamline Reporting and Tracking: Build a web dashboard where managers can review, categorize, and monitor incidents over time.

  • Improve Incident Resolution: Facilitate faster decision-making and follow-ups through real-time data access and comprehensive tracking.

Key Features & Functionalities

  • Mobile Incident Logging: A straightforward, form-based interface allowing floor supervisors to quickly log incidents with essential details, attachments, and timestamps.

  • Incident Categorization & Tagging: Supervisors can categorize incidents by type, urgency, and location, aiding in easy filtering and tracking.

  • Real-Time Web Dashboard: A robust dashboard for viewing summaries, filtering metrics, and drilling down into cases for detailed insights.

  • Automated Notifications: Customizable notifications for critical incidents, ensuring immediate attention from relevant team members.

  • Comprehensive Reporting Tools: Managers can generate and export reports on incident trends and resolutions, aiding data-driven decisions.

  • Cross-Platform Data Syncing: Ensures real-time data updates across mobile and web platforms.

Design Details

  • Quick-Entry Mobile Interface: Large touch targets and minimal fields for rapid data entry, even in dynamic environments.

  • Data Visualization: Web dashboard visualizations, including charts and trends, help managers identify patterns and high-frequency issues.

  • Intuitive Filtering & Sorting: Sort incidents by time, severity, or type to prioritize critical issues.

  • Responsive & Accessible Design: Fully responsive web interface, compatible with tablets, desktops, and mobile devices.

My Role

I led the end-to-end design process for the Incident Logging & Reporting Tool, which involved the following key responsibilities:

  • User Research: Conducted in-depth research to understand floor supervisors' needs, pain points, and workflows in gaming venues.

  • Wireframing: Created initial layouts and structures for both the mobile and web interfaces, focusing on efficient incident logging and streamlined reporting.

  • Prototyping: Developed interactive prototypes to validate functionality, gather user feedback, and ensure ease of use in fast-paced environments.

  • UI/UX Design: Designed a visually engaging, accessible, and responsive interface optimized for both mobile and web, focusing on data readability and user efficiency.

  • Data Visualization: Integrated data visualizations, including charts and trend indicators, to help managers quickly interpret incident data and identify recurring issues.

  • Responsive Design: Ensured consistency and usability across devices, with interfaces adaptable for various screen sizes from mobile to desktop.

  • Usability Testing: Conducted iterative testing with end-users to refine the interface and ensure that all functions were intuitive, efficient, and user-friendly.
