Edraw Application - iPad Application with Screencast for Gaming Floor

Seamless Draw Management with Real-Time Transparency for Gaming Venues

Overview

Designed specifically for gaming venues, this application simplifies the process of conducting draws on the gaming floor. The tool supports single and multiple winner selections and leverages screencasting for a transparent and engaging experience. The application empowers floor supervisors with an intuitive and visually appealing interface to manage live draws seamlessly.

Project Goals

  • Streamline Draw Process: Provide a fast and intuitive system that simplifies the selection and announcement of winners on the gaming floor.

  • Enhance Transparency: Implement screencasting capabilities for live viewing, ensuring transparency and fairness in draw events.

  • Ensure Ease of Use: Design a user-friendly interface that enables supervisors to manage draws efficiently, with minimal training required.

Key Features & Functionalities

  • Single and Multiple Winner Selection: Easily select either a single or multiple winners, with customizable parameters to suit different event types.

  • Screencasting for Transparency: Provides real-time display of the draw process on large screens, enhancing trust and engagement among participants.

  • Intuitive Draw Controls: Accessible controls allow supervisors to manage the draw with minimal steps, focusing on a smooth and efficient flow.

  • Customizable Draw Settings: Set specific rules, such as eligibility requirements and draw intervals, directly within the app.

  • Clear Visual Design: Bold typography and high-contrast colors ensure that information is easy to read and follow, even in dynamic environments.

Design Details

  • Optimized for iPad: Built for iPad with responsive elements for an engaging experience on larger screens, ideal for visibility on the gaming floor.

  • Real-Time Updates: Ensures instant refreshes and displays updates as the draw progresses, keeping audiences and supervisors informed.

  • User-Centric Interface: Designed with large buttons and simple navigation to facilitate quick actions in high-paced settings.


Results & Impact

The implementation of this tool significantly improved the draw process on the gaming floor, reducing time spent and increasing participant engagement. The screencast feature ensured transparency, fostering greater trust in the draw outcomes.

  • User Engagement: The transparency of screencasting led to a 40% increase in participant engagement during draw events.

  • Time Efficiency: Supervisors reported a 70% reduction in time taken to conduct draws, thanks to the streamlined process.

  • Positive Feedback: Received favorable feedback from both participants and supervisors, emphasizing ease of use and the engaging interface.

My Role

I managed the full design process for this project, covering:

  • User Research: Gathered insights from floor supervisors to understand their workflow and draw process needs.

  • Wireframing: Developed initial layout concepts that prioritized intuitive navigation and ease of use.

  • Prototyping: Built interactive prototypes for testing and validation, ensuring functionality and usability.

  • UI/UX Design: Created a cohesive design, including clear typography and high-contrast elements for readability in gaming environments.

  • Screencasting Integration: Designed and implemented screencasting features, allowing real-time display of draws to engage audiences.

  • Usability Testing: Conducted testing sessions to refine the interface based on user feedback, focusing on speed and ease of navigation.
